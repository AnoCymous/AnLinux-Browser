A Internet solution for [AnLinux](https://github.com/EXALAB/AnLinux-App) projects.

## Requirement
- Ubuntu >= 16.04
- Any Linux distribution without [the Install Shell Script](install.sh)
##### (below will be install by [install.sh](install.sh))
- Python >= 3.5
- gir1.2-webkit-3.0
-  python-gi

## Installation
    ./install.sh

## Usage
    ./browser.py
